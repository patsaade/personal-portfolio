# Examples (not deployed)

Files in this directory are reference templates only. They live **outside**
`src/content/`, so Astro never builds pages for them, and they're listed in
`.vercelignore` so they aren't uploaded to Vercel either.

- `labs/szechuan-sauce-memory.mdx` — a sample lab writeup showing the expected
  structure (Challenge Summary → Approach → Key Findings → Timeline → IOCs →
  Lessons Learned). To publish a real lab, copy it into `src/content/labs/`
  and edit the frontmatter + body.
